# Mushroom-Classification
This project classifies mushrooms as edible or poisonous using machine learning. Models like Random Forest, XGBoost, Logistic Regression, and Decision Tree achieved up to 99% accuracy. A Tkinter GUI lets users input mushroom features and instantly view predictions in a simple, interactive way. 
# 🍄 Mushroom Classification with Machine Learning & Tkinter

## 📖 Overview
This project predicts whether a mushroom is **edible or poisonous** using machine learning algorithms. It integrates a **Tkinter-based GUI** so users can input mushroom features and instantly see predictions.

## 📊 Dataset
- Source: Kaggle (Mushroom Classification Dataset)  
- Features: Cap shape, cap color, odor, gill size, habitat, etc.  
- Target: Edibility (edible or poisonous)

## ⚙️ Algorithms Used
- Decision Tree  
- Logistic Regression  
- Random Forest  
- XGBoost  

## 🎯 Results
- Achieved up to **99% accuracy** with Random Forest and XGBoost.  
- Models evaluated using accuracy, precision, recall, and confusion matrix.  

## 🖥️ GUI Features
- Built with **Tkinter** for easy interaction.  
- Simple interface for non-technical users.  
- Instant predictions based on user input.
